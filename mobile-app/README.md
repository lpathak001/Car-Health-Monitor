# Car Health Monitor - Mobile App

React Native mobile application with 4 feature units.

## Units

1. **Dashboard** - Health visualization, charts, gauges
2. **Alerts** - Notifications and alert management
3. **Vehicle** - Vehicle profile and management
4. **Profile** - User settings and preferences

## Quick Start

```bash
npm install
npm start
```

## Structure

- `/dashboard` - Dashboard unit with health score and charts
- `/alerts` - Alerts unit with notification list
- `/vehicle` - Vehicle management unit
- `/profile` - User profile and settings unit

## Features

- Bottom tab navigation
- Health score gauge
- Sensor data visualization
- Alert notifications
- Vehicle management
- User profile settings
