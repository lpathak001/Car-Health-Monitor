## 🚀 How to Use

### Option 1: Present from Markdown
```bash
# Install reveal-md
npm install -g reveal-md

# Run presentation
reveal-md CUSTOMER_DEMO_SLIDES.md

# Open browser to http://localhost:1948
# Use arrow keys to navigate
```

### Option 2: Export to PowerPoint
```bash
# Install Marp
npm install -g @marp-team/marp-cli

# Convert to PowerPoint
marp CUSTOMER_DEMO_SLIDES.md --pptx -o customer-demo.pptx

# Open in PowerPoint and customize
```

### Option 3: Export to PDF
```bash
# Using Marp
marp CUSTOMER_DEMO_SLIDES.md --pdf -o customer-demo.pdf

# Or using Pandoc
pandoc CUSTOMER_DEMO_SLIDES.md -o customer-demo.pdf
```

### Option 4: Manual Creation
1. Open PowerPoint or Google Slides
2. Copy content from CUSTOMER_DEMO_SLIDES.md
3. Format each slide
4. Add company branding
5. Customize as needed
