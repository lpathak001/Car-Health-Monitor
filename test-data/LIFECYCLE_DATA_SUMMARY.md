## 🚀 Loading the Data

### Python
```python
import json

with open('test-data/synthetic_data_lifecycle_25years.json', 'r') as f:
    data = json.load(f)

print(f"Total readings: {len(data['readings']):,}")
print(f"Total miles: {data['statistics']['total_miles']:,}")
print(f"Final health: {data['statistics']['final_health_score']}%")
```

### Analyze Maintenance
```python
maintenance = data['maintenance_events']
total_cost = sum(m['cost'] for m in maintenance)
print(f"Total maintenance cost: ${total_cost:,}")

# Group by type
from collections import Counter
types = Counter(m['type'] for m in maintenance)
for mtype, count in types.most_common():
    print(f"{mtype}: {count} times")
```

### Track Health Over Time
```python
import pandas as pd

# Sample readings (every 1000th for performance)
sampled = data['readings'][::1000]
df = pd.DataFrame(sampled)

# Plot health score over time
df['timestamp'] = pd.to_datetime(df['timestamp'])
df.plot(x='timestamp', y='health_score', 
        title='Vehicle Health Over 25 Years')
```
